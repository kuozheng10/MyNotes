## Gmail Automation v2.0 重寫

### 架構改變
- **Label 從 8 個精簡到 4 個**：重要文件 / 銀行金融 / 購物旅遊 / 電子報
- **重要文件優先保護**：主旨含帳單/收據/確認書等 → 永不 trash
- **新增函數**：
  - `cleanupInbox()` — 一次性清 90天~1年內 inbox（手動跑一次）
  - `migrateLabels()` — 舊 label 遷移到新 label（手動跑一次）
  - `deleteEmptyLabels()` — 刪除空 label
  - `isThreadStarred()` — 修 GmailThread 無 isStarred() 的 bug
- **未讀也有寬限天數**（已讀的 2 倍），不再永遠不動
- 規格書：`gmail-automation-spec.md`

### 直接用 Gmail API 做的事
- 刪除 8 個空 label（舊版遺留）
- 「登入通知信」label 下 276 則全部移垃圾桶，label 刪除
- Token：`~/.claude/google-token.json`（gmail.modify + drive scope）
