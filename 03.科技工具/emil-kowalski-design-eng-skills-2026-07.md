# Emil Kowalski：給 Coding Agent 的設計工程 Skills

> **一句話**：前 Vercel / Linear 設計工程師（Sonner、Vaul 元件作者）把多年 UI/動畫審美經驗包成 Skills，讓 Codex / Claude Code / Cursor 寫 UI 動畫時更接近資深設計工程師的判斷。

---

## 核心理念

> "AI doesn't replace such expertise, it amplifies what you can get out of it."

這些 skills 專門解決 AI coding agent 常犯的設計錯誤：選錯緩動函數（easing）、用不恰當的視覺元素等——AI 不是不會做動畫，是沒有品味判斷力，這套 skills 就是把品味判斷力外部化。

## 三個 Skills

| Skill | 用途 |
|-------|------|
| **emil-design-eng** | 主要技能，涵蓋動畫與設計建議 |
| **review-animations** | 依特定規則嚴格審查動畫實作 |
| **animation-vocabulary** | 教你用精確詞彙跟 AI 溝通動畫需求，講對詞才能要到對的效果 |

## 安裝方式

npx skills@latest add emilkowalski/skills
