## 這是什麼

社群策展的 GPT-Image-2 高品質 prompt 集合，含輸出圖片示範。
GitHub: https://github.com/EvoLinkAI/awesome-gpt-image-2-prompts

## 四大類別

| 類別 | 數量 | 代表內容 |
|------|------|---------|
| Portrait & Photography | 101 cases | 電影感人像、復古修復、編輯風格攝影 |
| Poster & Illustration | 167 cases | 旅遊海報、角色設計、產品廣告、資訊圖表 |
| Character Design | - | 遊戲/動漫角色表，多角度一致風格 |

---

## 特點

- 支援 11 種語言（含繁中/簡中）
- 每個 case 附輸出圖 + prompt 細節（光線、風格、色調）
- 持續更新（2026/4 仍有新增）
- 社群貢獻，附 Twitter 創作者標記

---

## 對派哥的意義

- My Wallet Trip / 個人專案需要封面圖、UI 示意圖時，可直接抄 prompt 結構
- 旅遊類、UI 類 prompt 最實用
- 學習 prompt 寫法：光線條件、藝術風格、鏡頭語言怎麼描述

## 是否包 skill？

不建議。這是靜態資源庫，直接查 GitHub 或 MyNotes 即可，不需要包成 skill。
若之後有大量圖像生成需求，再考慮做一個「圖像 prompt 快速查詢 skill」。
