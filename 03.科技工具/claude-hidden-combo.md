## 核心概念

> 三個 Claude 功能單獨用都還好，組合起來才真的嚇到你。
> 零程式碼，讓 AI 每天早上自動出門查資料、記下來，你回來問它要答案。

## 三個功能

### 1. 瀏覽器自動抓取（Claude in Chrome）
- 安裝 Chrome 擴充功能
- 可指定競品網站抓取首頁熱銷產品與價格
- 可同時分析多家競品，整理成比較表格

**範例：**
咖啡豆售價 + 促銷分析 → 多家競品比較表

### 2. 排程自動執行（Cowork Schedule）
- 設定固定時間自動執行任務（如每天早上）
- **注意**：需要電腦開著 + Claude Desktop 在背景執行才會觸發

### 3. Memory 記憶累積
- 資料跨 session 保留，隨時問隨時有答案
- **注意**：需在同一個 Project 內操作，Memory 才會跨 session 保留

## 對你有用嗎？

**有用的情境：**
- 定期追蹤競品/市場資訊，不想手動查
- 累積研究資料，讓 Claude 變成你的專屬資料庫
- 用 Claude Desktop 做輕量自動化，不想寫程式

**限制：**
- 排程需要電腦開著（不是 server-side，是本機執行）
- Memory 限 Project 內，跨 Project 不共享
- 用的是 Claude Cowork（Desktop app），不是 Claude Code
