## 這是什麼

利用 **NVIDIA Build 平台**提供的免費 API 額度，調用頂尖大模型（MiniMax M2.7、GLM 5.1），整合進 OpenClaw（龍蝦）等工具，實現零成本高效能 AI 互動。

來源：電腦王阿達 koc.com.tw

---

## 核心重點

- 230B 參數的 MiniMax M2.7 對個人硬體要求極高，NVIDIA API 讓一般用戶以雲端享受企業級效能
- **無需信用卡**，流程簡單，目前是體驗最新模型的最優路徑
- API 整合 > 網頁介面：解鎖 Agent 功能、自定義工作流、更低延遲

---

## 兩個主要模型

| 模型 | 特性 |
|------|------|
| MiniMax M2.7 | 自我演化能力，指令遵循/工具調用/複雜推理卓越 |
| GLM 5.1 | 國產中堅，中文語境理解、多模態任務領先 |

---

## 申請步驟

1. 訪問 **NVIDIA Build** 官網
2. 註冊/登入（只需 Email，不需信用卡）
3. 模型庫選擇 MiniMax M2.7 或 GLM 5.1
4. 點擊「Get API Key」取得金鑰

## 整合 OpenClaw

1. 打開 OpenClaw 設置
2. 填入 NVIDIA API 代理網址（End-point）
3. 貼上 API Key，選對應模型名稱

---

## 關鍵洞見

- **掌握「管道」比掌握「介面」更重要**：學會 API + 第三方工具，避免被單一平台綁架
- **資訊差即競爭力**：NVIDIA 免費額度有時效性，及時行動才能以最低成本保持技術優勢
- 不要侷限於 ChatGPT/Claude，MiniMax 在中文創意寫作、長文本理解已能分庭抗禮

---

## 對派哥的意義

- OpenClaw（龍蝦）= 派哥的 Claude Code，直接可用此方法擴充模型選擇
- 免費 API 額度適合測試 MiniMax/GLM 在特定任務的表現，再決定是否付費採用
- 建議：定期查 NVIDIA Build 有無新模型上線
