## 摘要

> macOS menu bar app，右上角小圖示顯示各 AI 服務 token 用量與配額進度條，本地計算不上傳。

## 功能

- 支援 Claude、Gemini、Codex、Cursor、Copilot 等 10+ 家 AI 服務
- 每週 / 每月用量 + 重置倒數計時
- 本地 30 天費用追蹤
- 服務異常警示
- 隱私優先：只讀本機 cookie / log，不掃全磁碟

## 適用場景

- 有訂閱 **Gemini Advanced**（Google One AI Premium）→ 可監控配額
- 有訂閱 Claude Pro / Max → 可監控配額
- **不適用**：My Wallet Trip 的 Gemini API key（Google AI Studio 直打 API，不走網頁帳號）

## 安裝

從 GitHub Releases 下載 `.dmg` 安裝，開啟後針對各 provider 登入授權即可。
