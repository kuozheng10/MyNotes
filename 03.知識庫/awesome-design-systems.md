## 這是什麼

開源策展清單，收錄 55 個大廠的設計系統，20K star。
不用買設計課，直接看大廠怎麼做。

收錄包含：Google Material、Apple HIG、Microsoft Fluent、Airbnb、Shopify、GitHub、Stripe 等。

---

## 如何使用

### 場景 1：做 UI 時不知道某元件怎麼設計
```
1. 想做按鈕/表單/導航 → 開 repo 找同類型產品的設計系統
2. 例如做 SaaS 後台 → 看 Stripe Dashboard / GitHub Design System
3. 例如做消費者 App → 看 Airbnb / Apple HIG
```

### 場景 2：和 Claude 一起做 UI
```
派哥說：「參考 Stripe 的設計邏輯做一個付款確認頁」
我去找 Stripe Design System 的規範 → 產出更貼近真實大廠風格的 UI
```

### 場景 3：學設計思維
每個設計系統都有說明「為什麼這樣設計」，比教學課程更真實。
