## 一句話核心

> 酒店會籍之間可以互相「配對」升級——用一個高卡觸發另一家的高卡挑戰或直配，省去住宿累積的時間。

---

## 信用卡直配通道

| 信用卡 | 可配對會籍 |
|--------|-----------|
| **MasterCard Travel Rewards（萬事達）** | Wyndham（開局卡）、GHA（開局卡）、I Prefer、Brilliant by Langham（附一張升套券）、Centra The 1、ASR 等 |
| **Visa 無限卡** | GHA 鈦金卡（開局卡）— 找 VISA 秘書申請 |

## 各酒店配對攻略

### Radisson（麗笙）
- 用其他家酒店高卡或航司寄信配對
- 大多給白金（偶有 VIP）
- 📩 `loyaltypartnerupgrade@radissonhotels.com`

### Best Western（最佳西方）
- 用 GHA 會籍配對（客服可能要求住宿記錄）
- 持有後可再配 **Choice 白金**，也可嘗試配 Scandic

### Choice 白金（精選酒店）
- 可用 BestWestern / Marriott / Hyatt / Hilton / IHG 配對

### I Prefer 鈦金卡
- 亞太區 MasterCard Redemption 配對
- 歐洲區 MasterCard：住一晚升鈦金 + 贈 25,000 分
- 歐區卡參考：Curve（537590）、匯豐卓越（519958）、巴克萊 Avios（530128）

### ASR（雅思閣）
- Hilton / Marriott / IHG / Accor 等高卡可配
- MasterCard 指定活動頁也可配對
- 部分機場提供免費貴賓室使用次數

### Thon+
- 用 Hilton / Radisson 配對
- 持有後可再配 **Scandic**
- 配對時間較長（曾有半年才拿到的案例）
- 目前須入住一次才算完成挑戰

### Hilton（希爾頓）
- 滴滴打車體驗 V8：取得鑽卡 120 天，住 12 晚（挑戰可得 2 年）
- 大陸星巴克鑽星會員可配對鑽挑
- 一般挑戰可觸發 Choice / BestWestern / Marriott / Accor / IHG 等配對

### Scandic
- 可用 Thon+ / Strawberry / Sokkos / Elite 配對（成功率較高）
- BestWestern / I Prefer / Hilton 有時被拒
- 需 D-Number，找客服開挑戰（盧小小），住 4 晚拿 Lv5（可直配 SAS 金卡）
- 📩 `member@scandichotels.com`

### Dusit Gold（杜斯特）
- 可用 BestWestern / Marriott / Hyatt / Hilton / IHG / Accor 配對
- 須提供一個未來住宿訂單編號（可預訂後配完再取消）

### Wyndham（溫德姆）
- MasterCard 可配對
- 其他酒店高卡也可配

### Sonesta
- 拿高卡寄 Email 可配對

### Marriott（萬豪）白金
- 電話客服取得 16 晚挑戰（可能需要先入住一次）
- 飛豬 F4 / 淘寶 88VIP / 支付寶黑鑽：取得 8 晚白金挑戰

### IHG（洲際）
- 購買洲際大使取得白金
- 持有東北亞／東南亞各國 MasterCard 可直配 IHG 金/白金

### Accor（雅高）
- 需購買 A+ 及臻享卡，最高疊加到 50 晚（距白金差 10 晚）

## 配對鏈示意（常見路徑）

```
MasterCard → GHA → 配 BestWestern → 配 Choice 白金
MasterCard → I Prefer → 配對歐區升鈦金
Hilton 鑽 → 配 Radisson 白金 → 配 Thon+ → 配 Scandic Lv5 → 直配 SAS 金
```
