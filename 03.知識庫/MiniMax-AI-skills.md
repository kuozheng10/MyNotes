## 摘要

> MiniMax 開源的 AI 編碼 skills 套件，支援 Claude Code / Cursor / Codex，提供生產級開發建議。

## 內容

MiniMax-AI 開發的開源專案，提供結構化開發指南給 AI 編碼工具使用。包含 14 個技能模組，MIT 授權。

## 重點摘錄

- 支援工具：Claude Code、Cursor、Codex、OpenCode
- **前端/全棧**：React、Next.js、Tailwind CSS、Framer Motion、GSAP
- **行動開發**：Android (Kotlin/Jetpack Compose)、iOS (UIKit/SwiftUI)、Flutter、React Native
- **圖形**：GLSL 著色器
- **文件生成**：PDF、PowerPoint、Excel、Word
- **AI 多模態**：MiniMax API 整合（語音、音樂、影片、圖像生成）
- 附本地驗證腳本確保品質
