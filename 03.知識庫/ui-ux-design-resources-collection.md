## 這是什麼
這是一份針對 UI/UX 設計師及開發者整理的資源清單，旨在提供高品質的落地頁、SaaS 網站、行動應用與字體設計參考，協助使用者在使用 Claude 等 AI 工具輔助設計時能產出更具質感的介面。

## 核心概念
*   **視覺參考庫**：包含 Landing Pages (landing.love) 與 SaaS 網站 (saaspo.com) 的深度案例研究。
*   **設計資源鏈結**：整合了綜合設計庫 (curations.supply) 與高品質開源字體 (uncut.wtf)。
*   **AI 驅動原型**：利用 sleek.design 等 AI 工具快速建構行動端應用程式原型。

## 使用方法 / 快速啟動
1.  **風格定位**：在請 Claude 生成網頁或 App 介面前，先從 saaspo 或 landing.love 挑選符合產品調性的風格。
2.  **提示詞強化**：將挑選出的網站特點（如：極簡主義、高對比色、特定字體排版）作為 Claude 的設計提示詞。
3.  **字體應用**：參考 uncut.wtf 挑選現代感字體，應用於 Web UI 的 CSS 樣式中。

## 對派哥的啟示
派哥在台灣開發自動化 AI 工具時，良好的 UI/UX 是決定產品專業度的關鍵。透過這些資源網站，派哥可以將現代 SaaS 的設計邏輯餵給 Claude，讓自動化工具不再只是冷冰冰的後端腳本，而是擁有國際水準的視覺介面。特別是針對 SaaS 類型的自動化服務，參考 saaspo.com 的佈局能顯著提升台灣用戶對 AI 工具的信任感與操作體驗。
