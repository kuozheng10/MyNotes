# 滙豐旅人卡積分兌換核心 Tips

## 基本架構

**滙豐旅遊積分** → 可轉 16 家航空 + 3 家飯店（含雅高 Accor）

---

## 最佳兌換路徑（倍率排名）

### 路徑 A：旅遊積分 → 雅高 → 澳洲航空 QF ★最高 1.875x★
```
4,000 旅遊積分 → 6,000 雅高積分 → 7,500 QF 哩程
放大倍率：1.875x
```
- QF 屬寰宇一家，可兌換 **日航、國泰** 等 OW 航班
- 台北—大阪 商務艙來回：只需 **33,600 旅遊積分**（vs 亞萬需 64,000 哩）

### 路徑 B：旅遊積分 → 雅高 → FB（法荷航藍天飛行）= 1.5x
```
2,000 旅遊積分 → 3,000 雅高積分 → 2,000 FB 哩程
放大倍率：1.5x（在 +50% 活動期間）
```
- FB 屬天合聯盟，可兌換 **華航** 等 SkyTeam 航班
- 台北—溫哥華 商務艙來回：只需 **83,500 旅遊積分**（vs 華航哩程需 160,000）

### 路徑 C：旅遊積分 → 雅高 → IB → BA/QR = 1.5x（但 QR 開票超划算）
```
3,000 旅遊積分 → 4,500 雅高積分 → 3,000 IB 哩程 = 3,000 BA/QR Avios
放大倍率：1.5x
```
- IB/BA/QR Avios 可互轉
- QR 開票率極低：台北—大阪 商務艙來回僅需 **33,000 Avios + 稅 1,430 元**
- 換算只需 **22,000 旅遊積分**（vs 亞萬需 64,000 哩）
