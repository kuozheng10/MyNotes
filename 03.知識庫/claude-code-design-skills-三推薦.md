## 三個推薦的設計 Skills

### 1. Frontend Design Skill（Anthropic 官方，136,000 ⭐）
- **來源**：https://github.com/anthropics/skills/tree/main/skills/frontend-design
- **痛點**：以前叫 AI 做網頁，每次都長一樣——Inter 字型、同樣卡片佈局
- **用途**：強迫 Claude 跳出制式 AI 感，設計輸出每次不同
- **用法**：給風格關鍵字（極簡主義、復古、...）而非直接說「幫我設計 XX 頁面」

### 2. UI/UX Pro Max（GitHub 7 萬 stars）
- **來源**：https://github.com/nextlevelbuilder/ui-ux-pro-max-skill
- **用途**：設計寶典，含配色、風格、字體組合
- **用法**：描述產品細節越詳細越好，讓它推薦整體設計系統

### 3. Web Design Guidelines（Vercel 出品）
- **來源**：https://github.com/vercel-labs/agent-skills/tree/main/skills/web-design-guidelines
- **用途**：檢查設計細節（對比度、按鈕大小、間距）
- **用法**：設計完丟程式碼或截圖給它審查

## 安裝方式

直接把 GitHub 網址貼給 Claude Code，請它安裝即可。
