## 摘要

> 55+ 個大廠品牌（Apple、Notion、Vercel、Stripe、Claude 等）的設計語言，全部打包成 DESIGN.md 格式。
> 複製一個到專案根目錄，告訴 AI「幫我做一個看起來像這樣的頁面」，AI 就能讀懂色系、字體、組件規範，直接生成像素級還原的 UI。
> ⭐ 18.8K stars，2.3K forks

## 怎麼用

1. 去 GitHub 選一個你喜歡的品牌 DESIGN.md（例如 Notion、Vercel、Linear）
2. 複製到你的專案根目錄
3. 開 Claude Code，說：「參考 DESIGN.md，幫我建立一個登入頁面」

## 評估：對派哥有用嗎？

**有用情境：** 做 My Wallet、cc_processor 等工具的 UI 時，想要乾淨有設計感的頁面。
直接拿 Notion 或 Linear 的 DESIGN.md 丟進去，效果比從頭說「做個深色簡潔的 UI」強很多。

**不需要包成 skill**：用法太簡單，就是複製一個 md 檔到專案根目錄。
需要時去 GitHub 抓，不用常備。
