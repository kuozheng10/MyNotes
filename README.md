# 派哥知識庫

Obsidian vault，存放閱讀筆記、技術文章、有用連結。

## 目錄結構

| 資料夾 | 用途 |
|--------|------|
| 01.銀行金融 | 信用卡、銀行、保險相關 |
| 02.投資理財 | 股票、基金、ETF |
| 03.科技工具 | AI、程式、工具教學 |
| 04.旅遊 | 旅遊攻略、訂房、航班 |
| 05.學習閱讀 | 電子報、課程、書摘 |
| 06.生活雜記 | 其他生活相關 |
| 99.Templates | 筆記模板 |

## 新增筆記

1. 複製 `99.Templates/note-template.md`
2. 填入 title、url、tags、date
3. 存到對應資料夾

## 自動備份

每小時自動 git commit + push 到 GitHub（由 launchd 排程）。
